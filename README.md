## 🧠 Funcionalidades

- ➕ Añadir nuevas tareas  
- ✅ Marcar tareas como completadas  
- 🗑️ Eliminar tareas  
- 🔍 Filtros: todas / activas / completadas  
- 💾 Persistencia con LocalStorage  
- 📊 Contador de tareas  
- 🌙 (Opcional) Modo oscuro  

---

## 🛠️ Tecnologías utilizadas

- ⚛️ React  
- ⚡ Vite  
- 🎨 CSS  
- 💾 LocalStorage API  
- 📦 JavaScript (ES6+)

## ⚙️ Instalación y ejecución

Clona el repositorio:

```bash
git clone https://github.com/tuusuario/taskify.git
```

Entra al proyecto:

```bash
cd taskify
```

Instala dependencias:

```bash
npm install
```

Inicia el proyecto:

```bash
npm run dev
```
